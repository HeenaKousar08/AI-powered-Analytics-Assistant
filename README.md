# 🤖 AI-Powered Analytics Assistant

A **smart, locally running data analysis tool** that allows users to upload datasets and interact with them using AI. This project combines **EDA, visualization, and LLM-powered insights** into a single intuitive dashboard.

---

## 🚀 Features

* 📂 Upload CSV / Excel datasets
* 📊 Automated **Exploratory Data Analysis (EDA)**
* 📈 Generate **interactive visualizations**
* 🧠 Ask questions using **AI-powered data agent**
* 🧹 Auto-clean datasets (missing values, duplicates, etc.)
* 📄 Generate **professional PDF reports**
* 🔒 100% **Local processing (privacy-friendly)**

---

## 🖥️ Tech Stack

* **Frontend/UI:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly, Matplotlib, Seaborn
* **AI Engine:** Ollama (Mistral / LLaMA models)
* **PDF Generation:** FPDF

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ai-analytics-assistant.git
cd ai-analytics-assistant
```

### 2️⃣ Create Virtual Environment

```bash
conda create -n ai_env python=3.10
conda activate ai_env
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```

---

## 🧠 AI Setup (Ollama)

Make sure you have **Ollama installed**:

```bash
ollama run mistral
```

or

```bash
ollama run llama3
```
